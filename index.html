6.Difference between Window and Document?

Window: It is the root level element in any web page.All Global variables are defined on the window object only.For example:alert(),confirm() are the methods in window object.The window object is the topmost object of the DOM hierarchy. It represents a browser window or frame that displays the contents of the webpage. Whenever a window appears on the screen to display the contents of the document, the window object is created.

Document: It is the direct child of window object(DOM).The document object represent a web page that is loaded in the browser. By accessing the document object, we can access the element in the HTML page.
